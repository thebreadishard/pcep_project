# PCEP Project

Udacity PCEP project: **Transformational AI — Analyzing Ecommerce Large Datasets for Machine Learning**.

Uses pandas, seaborn, matplotlib, and pyarrow to transform, analyze, and visualize Amazon Reviews 2023 datasets.

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Open `Udacity - PCEP Project.ipynb` in VS Code and select the `.venv` kernel.
